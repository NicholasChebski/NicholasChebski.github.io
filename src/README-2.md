# Реализованные функции:

## create_vector()

### Что делает
Создаёт одномерный массив NumPy с числами от 0 до 9 включительно.

### Аргументы
У данной функции нет аргументов.

### Вывод
Возвращает одномерный массив чисел от 0 до 9 (numpy.ndarray).

### Ошибки
Данная функция не выдаёт ошибок.

## create_matrix()

### Что делает
Создаёт матрицу размера 5 x 5, заполненную случайными вещественными числами из диапазона [0, 1).

### Аргументы
У данной функции нет аргументов.

### Вывод
Возвращает матрицу размера 5 x 5 со случайными значениями (numpy.ndarray).

### Ошибки
Данная функция не выдаёт ошибок.

## reshape_vector(vec: np.ndarray)

### Что делает
Преобразует одномерный массив из 10 элементов в матрицу размера 2 x 5.

### Аргументы
- vec - входной массив типа np.ndarray, который должен быть одномерным и содержать ровно 10 элементов.

### Вывод
Возвращает преобразованный массив формы размера 2 x 5 (numpy.ndarray).

### Ошибки
- InputError - если vec не имеет тип numpy.ndarray
- InputError - если vec не является одномерным массивом
- InputError - если vec не содержит ровно 10 элементов

## transpose_matrix(mat: np.ndarray)

### Что делает
Выполняет транспонирование матрицы, то есть меняет строки и столбцы местами.

### Аргументы
- mat - входная матрица типа np.ndarray, которая должна быть двумерной.

### Вывод
Возвращает транспонированную матрицу (numpy.ndarray).

### Ошибки
- InputError - если mat не имеет тип numpy.ndarray
- InputError - если mat не является двумерным массивом

## vector_add(a: np.ndarray, b: np.ndarray)

### Что делает
Выполняет поэлементное сложение двух векторов одинаковой длины.

### Аргументы
- a - первый вектор типа np.ndarray, должен быть одномерным
- b - второй вектор типа np.ndarray, должен быть одномерным

### Вывод
Возвращает результат поэлементного сложения векторов (numpy.ndarray).

### Ошибки
- InputError - если хотя бы один из аргументов не имеет тип numpy.ndarray
- InputError - если хотя бы один из аргументов не является одномерным массивом
- InputError - если векторы имеют разную длину

## scalar_multiply(vec: np.ndarray, scalar: int | float)

### Что делает
Умножает все элементы вектора на заданное число.

### Аргументы
- vec - входной вектор типа np.ndarray, должен быть одномерным
- scalar - число типа int или float

### Вывод
Возвращает результат умножения вектора на скаляр (numpy.ndarray).

### Ошибки
- InputError - если vec не имеет тип numpy.ndarray
- InputError - если vec не является одномерным массивом
- InputError - если scalar не является числом

## elementwise_multiply(a: np.ndarray, b: np.ndarray)

### Что делает
Выполняет поэлементное умножение двух массивов одинаковой формы.

### Аргументы
- a - первый массив типа np.ndarray
- b - второй массив типа np.ndarray

### Вывод
Возвращает результат поэлементного умножения массивов (numpy.ndarray).

### Ошибки
- InputError - если хотя бы один из аргументов не имеет тип numpy.ndarray
- InputError - если массивы имеют разную форму

## dot_product(a: np.ndarray, b: np.ndarray)

### Что делает
Вычисляет скалярное произведение двух векторов.

### Аргументы
- a - первый вектор типа np.ndarray, должен быть одномерным
- b - второй вектор типа np.ndarray, должен быть одномерным

### Вывод
Возвращает скалярное произведение векторов (float | numpy.ndarray).

### Ошибки
- InputError - если хотя бы один из аргументов не имеет тип numpy.ndarray
- InputError - если хотя бы один из аргументов не является одномерным массивом
- InputError - если векторы имеют разную длину

## matrix_multiply(a: np.ndarray, b: np.ndarray)

### Что делает
Выполняет умножение двух матриц.

### Аргументы
- a - первая матрица типа np.ndarray, должна быть двумерной
- b - вторая матрица типа np.ndarray, должна быть двумерной

### Вывод
Возвращает результат умножения матриц (numpy.ndarray).

### Ошибки
- InputError - если хотя бы одна из матриц не имеет тип numpy.ndarray
- InputError - если хотя бы одна из матриц не является двумерным массивом
- InputError - если число столбцов первой матрицы не совпадает с числом строк второй

## matrix_determinant(a: np.ndarray)

### Что делает
Вычисляет определитель квадратной матрицы.

### Аргументы
- a - матрица типа np.ndarray, должна быть двумерной и квадратной

### Вывод
Возвращает определитель матрицы (float).

### Ошибки
- InputError - если a не имеет тип numpy.ndarray
- InputError - если a не является двумерным массивом
- InputError - если a не является квадратной матрицей

## matrix_inverse(a: np.ndarray)

### Что делает
Находит обратную матрицу для заданной квадратной матрицы.

### Аргументы
- a - матрица типа np.ndarray, должна быть двумерной и квадратной

### Вывод
Возвращает обратную матрицу (numpy.ndarray).

### Ошибки
- InputError - если a не имеет тип numpy.ndarray
- InputError - если a не является двумерным массивом
- InputError - если a не является квадратной матрицей
- LogicalError - если матрица является вырожденной

## solve_linear_system(a: np.ndarray, b: np.ndarray)

### Что делает
Решает систему линейных уравнений вида Ax = b.

### Аргументы
- a - матрица коэффициентов типа np.ndarray, должна быть двумерной и квадратной
- b - вектор свободных членов типа np.ndarray, должен быть одномерным

### Вывод
Возвращает решение системы (numpy.ndarray).

### Ошибки
- InputError - если a или b не имеют тип numpy.ndarray
- InputError - если a не является двумерным массивом
- InputError - если b не является одномерным массивом
- InputError - если a не является квадратной матрицей
- InputError - если размер b не совпадает с числом строк матрицы a

## load_dataset(path: str = "data/students_scores.csv")

### Что делает
Загружает CSV-файл и преобразует его содержимое в массив NumPy.

### Аргументы
- path - путь к CSV-файлу, должен быть строкой

### Вывод
Возвращает данные из CSV-файла (numpy.ndarray).

### Ошибки
- InputError - если path не является строкой
- InputError - если файл по указанному пути не найден
- InputError - если CSV-файл пуст

## statistical_analysis(data: np.ndarray)

### Что делает
Вычисляет основные статистические характеристики одномерного массива данных: среднее значение, медиану, стандартное отклонение, минимум, максимум, 25-й и 75-й перцентили.

### Аргументы
- data - одномерный массив типа np.ndarray

### Вывод
Возвращает словарь со статистическими характеристиками (dict): mean, median, std, min, max, percentile_25, percentile_75.

### Ошибки
- InputError - если data не имеет тип numpy.ndarray
- InputError - если массив data пустой
- InputError - если data не является одномерным массивом

## normalize_data(data: np.ndarray)

### Что делает
Выполняет Min-Max нормализацию массива по формуле:

(x - min) / (max - min)

### Аргументы
- data - входной массив типа np.ndarray

### Вывод
Возвращает нормализованный массив со значениями от 0 до 1 (numpy.ndarray).

### Ошибки
- InputError - если data не имеет тип numpy.ndarray
- InputError - если массив data пустой
- LogicalError - если все значения массива одинаковы
