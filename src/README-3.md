3. Демонстрация результатов выполнения:

		- Сайт: https://nicholaschebski.github.io/
4. Как был реализован деплой:
	-  Деплой на SourceCraft:
		1. Были созданы файлы ci.yaml и sites.yaml в папке .sourcecraft/
		2. В файле sites.yaml были прописаны ветка, из которой воркер клонирует репозиторий, и ветка, в которую ci пушит
		3. В файле ci.yaml были прописаны таски, которые воркер делает для деплоя сайта
		4. Также был создан файл requirements.txt для необходимых для установки модулей python (mkdocs)
	- Деплой на GitHub:
		1. Был создан файл deploy-release.yaml внутри папки .workflows/, находящейся внутри папки .guthub/
		2. Внутри файла deploy-release.yaml были прописаны таски, которые воркер делает для деплоя сайта
		3. Также был создан файл requirements.txt для необходимых для установки модулей python (mkdocs)
		4. Для воркера был создан токен, который хранится в secrets репозитория, который позволяет ему читать репозиторий и пушить в него
