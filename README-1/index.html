Full description

Site structure

There is a three pages: main page with all information about me, study page for all python labs and page for other projects not related to study. In "study" page there would be a list of labs updating with every lab made. You can click on it and it will open GitHub repository in next window of the browser.

Custom theme

There is a custom theme created on standart mkdocs theme. It changes colors to an emerald palette and hides elements like "search", "previous" and "next".
