Краткая сводка по работе

В данной лабораторной работе было реализовано обучение моделей логистической регрессии и случайного леса на обработанном обучающем датасете, были произведены замеры точности модели на обработанном тестовом датасете, а также была подобрана модель, получающая более высокие результаты точности, градиентный бустинг. Для моделей логистической регрессии, случайного леса и градиентного бустинга были составлены кривые ROC. Также был изложен алгоритм для интеграции обученной модели внутрь веб-сервиса.

    • Градиентный бустинг получил большую точность, чем случайный лес
  2. Пошаговый алгоритм: Предположим, что у нас есть обученная модель, которая может предсказывать, будет ли у человека дефолт по кредиту и веб-сервис банка, который позволяет взять кредит онлайн.
    1. При выборе на веб-сервисе опции "взять кредит", человек должен быть перенаправлен на страницу ввода данных, где он должен будет ввести все необходимые данные. (если их нет в системе заранее)
    2. После этого система подаёт данные на анализ модели, которая, например, даёт результат 1, если человеку можно выдать кредит, или 0, если человек не сможет выплатить кредит. В первом случае веб-сервис позволит человеку взять кредит. Во втором случае человеку будет отказано в выдаче кредита. Данная система позволит уменьшить нагрузку на банковских работников, что способствует уменьшению очередей в банках/на телефонной линии Если в постановке задачи имелась архитектура MVC, то во View добавится одна или несколько страниц с анкетой и страницы с результатом, а в Controller добавится анализ данных и определение возможности выдачи кредита (прогноз модели). Модель при этом будет частью Model.
