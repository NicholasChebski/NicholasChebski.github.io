Краткая сводка по работе

В данной лабораторной работе было реализовано обучение моделей линейной регрессии и случайного леса на обучающем датасете, были произведены замеры точности модели на тестовом датасете, а также была подобрана модель, получающая более высокие результаты точности, градиентный бустинг. Также были подобраны параметры для случайного леса, позволяющие получить небольшой отрицательный прирост в ошибках MAE и RMSE. Также был изложен алгоритм для интеграции обученной модели внутрь веб-сервиса.

  2. Пошаговый алгоритм: Предположим, что у нас есть обученная модель, которая может предсказывать (оценивать) стоимость недвижимости по показателям и веб-сервис компании, которая предоставляет услугу оценки недвижимости, который позволяет оценить недвижимость онлайн.
    1. При выборе на веб-сервисе опции "оценить", человек должен быть перенаправлен на страницу ввода данных, где он должен будет ввести все необходимые данные о недвижимости.
    2. После этого система подаёт данные на анализ модели, которая выдаёт цену, в которую оценивает эту недвижимость. Данный сервис будет полезен как для продавцов недвижимости, так и для покупателей, поскольку будет давать точную, справедливую и независимую оценку Если в постановке задачи имелась архитектура MVC, то во View добавится одна или несколько страниц с анкетой и страницы с результатом, а в Controller добавится анализ данных и определение стоимости недвижимости (прогноз модели). Модель при этом будет частью Model.
